# SIGCE

## SIGCE: Shapley Interactions-Guided Counterfactual Explanations for Regression Tasks

This repository contains the implementation of **SIGCE (Shapley Interactions-Guided Counterfactual Explanations)**, a novel framework for generating counterfactual explanations in regression tasks. It also includes the implementations of baseline methods, **DiCE** and **SVCE**, for comparison.
